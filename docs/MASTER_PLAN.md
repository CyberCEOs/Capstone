2. **Blue Team (Defender):** An LLM-integrated SIEM (ZySec-7B + Wazuh) that monitors logs in real-time to detect, report, and eventually counter-act adversarial actions.

3. **Autonomous Operations:** Demonstrate a full kill-chain (Recon -> Exploit -> Reporting) with minimal human prompt intervention.
